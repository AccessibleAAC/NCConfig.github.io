About netClé

Welcome to netClé

NetClé is a customizable user-friendly device that enables people with a range of mobility limitations to independently use a computer, tablet or similar device. At the heart of netClé is a custom hub containing an ATmega32U4 micro-controller which can accomodate a selection of input and output peripherals. These peripherals can be combined and configured in a variety of ways to allow the user to operate netClé by touch or movement (from a single digit, limb - or even their head), in order to perform mouse and keyboard functions that are difficult for the user to accomplish with standard input devices.

Some actions netClé can perform, via a USB or Bluetooth connection, include:

  • Move the cursor across the screen: up, down, left and right
  • Left and right mouse click
  • Scrolling up or down
  • Drag and drop
  • Keyboard actions (including key combinations)

Connecting netClé

The netClé hub peripheral ports are labelled from 0 to 3 and each netClé peripheral has a colour-coded base to indicate which port or ports it is compatible with. Peripherals are connected to the hub using the white 3.5mm male-to-male 4 conductor TRRS audio cables, included with the kit. Be sure that all cable connections are fully inserted into the hub and peripheral ports. Once the peripherals are connected, use the USB cable provided to connect the hub to a USB port on the computer displaying this web page. The top of the netClé should glow green after a few seconds.

You can configure netClé to meet your needs by using the netClé Configuration Tool.

Configuring netClé

After you have selected and connected the peripherals that are suitable for you, you can configure netClé to meet your particular needs by using the netClé Configuration Tool.

netClé Hub

The netClé hub contains a custom board based on an ATmega32U4 microcontroller.

  • Dimensions: 77mm x 68mm 23mm
  • Weight: 60g
  • Powered over USB: 5V
    • 50mA (without Bluetooth daughter board installed)
    • 100mA (with Bluetooth daughter board installed)

netClé can be configured to emulate a USB mouse and/or keyboard.

If an optional Bluetooth daughter board is installed, netClé can also emulate a Bluetooth mouse and/or keyboard.

Touch Sensor

  • An assistive switch requiring minimal pressure to activate
  • Operated by/with:
    • Arm movement
    • Moderate hand function
    • Good finger control, dexterity
    • Can be mounted, touched with cheek
  • Dimensions: 32mm x 33mm x 16mm
  • Weight: 10g
  • Compatible with yellow splitter

Dual Touch Sensor

  • An assistive switch requiring minimal pressure to activate
  • Dual sensors offer easy access to two functions concurrently
  • Operated by/with:
    • Arm movement
    • Moderate hand function
    • Some finger control, dexterity
  • Dimensions: 32mm x 57mm x 16mm
  • Weight: 16g
  • NOT compatible with yellow splitter

Mini Touch Sensor

  • A proximity switch requiring no pressure to activate
  • Operated by/with:
    • Arm movement
    • Moderate hand function
    • Good finger control, dexterity
    • Can be mounted, touched with cheek
    • Can be operated with a mouthstick
  • Dimensions: 19mm x 22mm x 15mm
  • Weight: 5g
  • Compatible with yellow splitter

Button Switch

  • A pressure switch requiring moderate touch to click and activate
  • Operated by/with:
    • Arm movement
    • Moderate hand function
    • Some finger control, dexterity
    • Can be mounted, touched with cheek
    • Can be operated with a mouthstick
  • Diameter: 59mm
  • Weight: 22g
  • Compatible with yellow splitter

Large Joystick

  • A standard Joystick that operates much like the joystick on a power wheelchair
  • Operated by/with:
    • Arm movement
    • Moderate hand function
    • Ability to grasp/grip
  • Base dimensions: 78mm x 105mm x 50mm (not including side flanges)
  • Base dimensions: 108mm x 105mm x 50mm (including side flanges)
    • Shaft and knob extend 60mm above the base
    • Knob diameter: 35mm
  • Weight: 285g
  • NOT compatible with splitter

Large Joystick with Palm Cradle

  • The cradle makes it possible for people with minimal or no grip strength to operate the joystick
  • Operated by/with:
    • Arm movement
    • Minimal hand function
  • Base dimensions: 78mm x 105mm x 50mm (not including side flanges)
  • Base dimensions: 108mm x 105mm x 50mm (including side flanges)
    • Shaft and cradle extend 40mm above the base
    • Cradle dimensions: 34mm x 133mm x 45mm
    • Inner cradle width: 99mm
  • Weight: 300g
  • NOT compatible with splitter

A rubber band can be looped through each eyelet and over the side of the cradle to prevent the cradle from rotating.

Mini Joystick

  • A small easy to use joystick that is operated by one or two fingers
  • Operated by/with:
    • Arm movement
    • Moderate hand function
    • Good finger control, dexterity
  • Base dimensions: 40mm x 51mm x 28mm
    • Handle extends 12mm above the base
  • Weight: 28g
  • NOT compatible with splitter

Mini Joystick with Extender Rod

  • The extender rod operated the mini joystick with less finger strength
  • Operated by/with:
    • Arm movement
    • Moderate hand function
    • Some finger control, dexterity
  • Base dimensions: 40mm x 51mm x 28mm
    • Handle extends 50mm above the base
  • Weight: 30g
  • NOT compatible with splitter

Gyro

  • Mount to a headband or hat, the gyroscope controls functions by moving the device left, right, up and down
  • Operated by/with:
    • Head movement: left, right, up, down
    • Attach to a user's arm
  • Dimensions: 25mm x 28mm x 17mm
  • Weight: 8g
  • netCle can only support 1 gyro peripheral
  • Compatible with green splitter when used with lightbox peripheral

Lightbox

  • Provides visual feedback to help the user identify the mode the peripheral is in, when the peripheral is set to perform multiple functions
  • Dimensions: 38mm x 122mm x 35mm
  • Weight: 42g
  • Compatible with green splitter

Splitters

  • Enables 2 peripherals to be plugged into the same port
    • Green splitter for gyro/lightbox
    • Yellow splitter for single switches and touch sensors
  • Dimensions: 30mm x 30mm x 14mm
  • Weight: 9g
  • Maximum 1 splitter per port (cannot be daisy-chained)

Switch Adaptor

  • Enables commercial switches with standard mono plugs to connect to netClé
  • Supports 1 or 2 commercial switches
  • Dimensions: 30mm x 30mm x 14mm
  • Weight: 9g
  • NOT compatible with yellow splitter
  • Maximum 1 adaptor per port (cannot be daisy-chained)
